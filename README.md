# CyberSafeBot (WPF Version)

## Overview

CyberSafeBot is a cybersecurity awareness chatbot developed in C# using Windows Presentation Foundation (WPF). The application is designed to educate users about common cybersecurity topics such as phishing, password safety, malware, and VPNs through an interactive chat interface.

This project was developed as part of a Programming POE assignment and demonstrates the use of object-oriented programming, user interaction, basic memory functionality, and graphical user interface design.

---

## Features

### User Interaction
- User enters their name before starting the chat.
- Personalized responses using the user's name.
- Interactive chatbot conversation through a graphical interface.

### Cybersecurity Awareness Topics
The chatbot provides information about:
- Phishing
- Password Security
- Malware
- Virtual Private Networks (VPNs)

### Memory Functionality
- Remembers cybersecurity topics discussed during the conversation.
- Allows users to ask about previously discussed interests.

### Sentiment Detection
The chatbot responds to user emotions such as:
- Happy
- Sad
- Worried
- Angry
- Confused
- Frustrated

### Graphical User Interface
- Built using WPF.
- Chat display area.
- User input textbox.
- Send button for interaction.

### Additional Features
- Welcome audio greeting.
- CyberSafe branding/logo.
- Randomized responses for more natural interaction.
